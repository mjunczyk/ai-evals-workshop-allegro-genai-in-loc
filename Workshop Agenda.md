# Workshop Agenda

## Evaluation Best Practices: Building Automated QA Pipelines for Agentic AI

Date: June 24, 2026
Time: 17:40 - 20:00 CEST (UTC+2) (140 minutes)

## Materials Checklist

- Access to the [LangSmith workspace](https://eu.smith.langchain.com/)

- Spreadsheet tool (Google Sheets or Excel)
- Workshop repository access

## 1. Welcome and Context (17:40 - 17:50, 10 min)

- Introductions and workshop goals
- Why evaluation matters in AI-assisted localisation
- Overview of today’s deliverables

## 2. Framework Briefing: Error Taxonomy and Severity (17:50 - 17:55, 5 min)

- Accuracy error subtypes used during annotation
- Severity levels and decision boundaries
- Acceptable vs. critical translation issues

## 3. Live Annotation Exercise (17:55 - 18:10, 15 min)

- Group annotation of sample e-commerce segments
- Alignment on annotation standards for the workshop

## 4. AI Evaluation Tool setup (18:10 - 18:50, 40 min)

- LangSmith setup (dataset, baseline prompt)
- Execute baseline experiment

## 5. Evaluation and Prompt Experimentation Run in LangSmith (18:50 - 19:35, 45 min)

- Configure evaluation criteria and run settings
- Execute structured evaluation
- Review metrics and representative failures
- Capturing prompt versions
- Iterative prompt refinement based on annotation goals

## 6. Results Review and Workflow Improvements (19:35 - 19:50, 15 min)

- Identify failure modes by category/severity
- Translate findings into prompt and process improvements
- Define next iteration priorities

## 7. Wrap-Up (19:50 - 20:00, 10 min)

- Open questions
- Recommended next steps for teams
- Reference resources and follow-up materials

## Expected Outcomes

- Annotated sample set aligned on taxonomy and severity labels
- At least one improved prompt iteration grounded in annotation feedback
- Initial evaluation run with documented failure modes
- Clear next-step plan for team-level rollout
